SCHNELLSTART
═══════════════════════════════════════════════════════════════════════

Windows-Benutzer:
-----------------
1. Doppelklick auf "START.bat"
2. Fertig! Die App öffnet sich im Browser.

Mac/Linux-Benutzer:
-------------------
1. Öffnen Sie das Terminal in diesem Ordner
2. Führen Sie aus: ./start.sh
3. Fertig! Die App öffnet sich im Browser.

ALTERNATIV: Manuelle Installation
═══════════════════════════════════════════════════════════════════════

1. Node.js installieren (falls nicht vorhanden):
   → Download: https://nodejs.org/

2. Terminal/Kommandozeile in diesem Ordner öffnen

3. Dependencies installieren:
   npm install

4. Entwicklungsserver starten:
   npm run dev

5. Browser öffnet sich automatisch auf http://localhost:3000
